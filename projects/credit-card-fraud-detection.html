Credit Card Fraud Detection using Machine Learning

Fraudulent credit card transactions are rare compared to legitimate ones, creating a severe class imbalance that makes them hard to detect reliably without the right modeling approach.

View on GitHub

1Problem Statement

Fraudulent credit card transactions are rare compared to legitimate ones, creating a severe class imbalance that makes them hard to detect reliably without the right modeling approach.

2Approach & Solution

Cleaned and preprocessed the transaction dataset, extracted and scaled relevant features, then trained and compared multiple classification models — including Logistic Regression and K-Nearest Neighbors — to identify fraudulent transactions. Evaluated all models using Precision, Recall, F1-score and ROC-AUC rather than raw accuracy, since accuracy is misleading on imbalanced fraud data, and selected the best-performing model for deployment.
